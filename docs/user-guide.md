## 日历页

日历页是主页面，展示月视图和当天的时间安排。

### 查看日程
- 左右滑动或点击箭头切换月份
- 日期下方的彩色圆点表示当天有预约
- 点击某个日期，下方显示当天的时间块

### 时间块说明
| 样式 | 含义 | 点击操作 |
|------|------|---------|
| 虚线边框 + "可预约" | 该时段空闲 | 跳转到预约页，自动填入时间 |
| 金色/粉色左边框 + 服务名 | 已有预约 | 弹出预约详情（客户、电话、备注） |
| 红色边框 + "已屏蔽" | 该时段被屏蔽 | 弹出操作面板，可取消屏蔽 |

### 重叠预约
如果两个预约时间有重叠：
- 两张卡片**并排显示**，各占一半宽度
- 重叠的时间边界显示为**红色**数字
- 点击任一卡片查看各自详情

## 添加预约

### 从日历页添加
1. 在日历页点击一个「可预约」时间块
2. 自动跳转到预约页，日期和时间已预填
3. 选择服务类型，结束时间自动计算
4. 填写客户信息，点击「确认添加」

### 从预约页添加
1. 切换到「预约」Tab
2. 点击右上角「+ 添加」按钮
3. 填写表单：
   - **服务类型**（下拉选择，来自设置里的服务列表）
   - **开始时间**（选择后结束时间自动按服务时长计算）
   - **结束时间**（可手动调整）
   - **客户姓名**（必填）
   - **手机号 / 微信号 / 备注**（选填）
4. 点击「确认添加」

### 自动计算结束时间
选择服务类型或修改开始时间后，结束时间会根据该服务的「预计时长」自动更新。你仍然可以手动修改结束时间。

## 预约列表

### 筛选预约
页面顶部有 4 个筛选标签：
- **全部** — 显示所有预约
- **待确认** — 新添加的预约，尚未确认
- **已确认** — 已确认的预约
- **已取消** — 已取消的预约

### 查看详情
点击预约卡片展开，显示：
- 手机号（点击可复制）
- 微信号（点击可复制）
- 备注信息

### 确认预约
展开一个「待确认」的预约，点击金色**「确认预约」**按钮。

### 取消预约
展开预约后，点击**「取消预约」**按钮，确认后状态变为已取消。

### 下拉刷新
在预约列表页下拉可刷新数据。

## 设置

### 营业时间
- **开始时间 / 结束时间** — 点击选择，决定日历页显示的时间范围
- **时间间隔** — 15分 / 30分 / 60分，影响可预约时段的划分粒度
- 默认：全天可约（00:00 - 24:00），30 分钟间隔

### 服务类型
每个服务包含：
- **名称** — 如「面部护理」「美甲设计」
- **颜色** — 从 12 色调色板选择
- **预计时长** — 15 / 30 / 45 / 60 / 90 / 120 分钟（单选）

操作：
- **添加**：点击「+ 添加」，填写信息后保存
- **编辑**：点击已有服务卡片，修改后保存
- **启用/禁用**：滑动右侧开关
- **删除**：向左滑动卡片，点击红色删除按钮

### 主题风格
两套主题可切换：
- **蜜桃** — 暖白底 + 玫瑰粉点缀（默认）
- **墨金** — 纯黑底 + 琥珀金点缀

点击主题预览圆即可切换，立即生效，下次打开仍保持。

### 版本号
页面底部显示当前版本号。
